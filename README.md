# BIGBEE-Open-Source-Intercom
The First Zero-Drain, No-Cloud Retrofit Intercom for analog 4+n systems. Features True UPS architecture (>7 months battery), proprietary "Magic OTG" offline config, and Zigbee/Matter Mesh readiness. Powered by ESP32-S3. Designed for privacy and longevity. Fully Open Source Hardware released under CERN-OHL-S v2.
